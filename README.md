# Real-Time Interpreter App (Expo)

Prototype app to record speech, transcribe with OpenAI Whisper API, and translate text using GPT-4o mini.



## Quick start
1. Download Expo Go
   https://expo.dev/go
3. Install dependencies:
   ```bash
   npm i
   ```
4. Create a file named `.env` in project root and add basic details.
5. Change IP address in `.env` to local IPv4, port 3000.
6. Run the app:

   cd backend
   ```bash
   npm install
   npm run dev
   ```
   Open new terminal:
   ```bash
   npx expo start --clear
   ```
